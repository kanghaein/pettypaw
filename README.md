
# pettypaw 🐾
🐕 일정을 공유할 수 있는 반려동물 캘린더 어플리케이션입니다.
### 개발기간

> - kminzy

### 개발환경
> `Java`
> `Android Studio`
> `Firebase`
### 기능
> 1. 회원가입 시 그룹장/그룹원 선택
> 2. 강아지들의 프로필 등록
> 3. 원하는 강아지 선택 후 필요한 일정 등록
> 4. 그룹 캘린더에서 강아지들의 일정 등록여부 확인
> 5. 그룹원끼리 강아지의 기본/상세 일정을 공유
> 6. 원하는 친구 초대 및 그룹 관리
